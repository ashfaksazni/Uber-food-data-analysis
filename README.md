# Uber Food Data Analysis

This project analyzes Uber food order data using Python in Jupyter Notebook to uncover insights about customer behavior, sales performance, and ratings.

## Tools & Technologies

* Python
* Jupyter Notebook
* Pandas
* NumPy
* Matplotlib
* Seaborn

## Project Objectives

* Explore and understand the Uber food dataset
* Clean and prepare the data for analysis
* Calculate key performance indicators (KPIs)
* Identify patterns in ratings, orders, and sales

## Key Analysis Performed

* Data cleaning and preprocessing
* Metadata exploration
* Total sales calculation
* Average order value
* Average rating analysis
* Rating distribution

## Project Outcome

The analysis provides insights into customer satisfaction, order trends, and overall performance of the Uber food service.
